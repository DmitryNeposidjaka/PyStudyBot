#What is it

This bot will helps you to study python while practice on our test.

#Get started

At first open bot and /start using it, we will register you and then you can start testing.

If you just want check your knowledge in python you need to select /test. Then you need to select category of tasks, now we have only four, but you always can ask to add more just contact me. And the last one is select tasks count. So there are your tasks you can use /stop or /next, you know what it is.

If you don`t like our tasks, or you want to share your experience, just /add one more. At first you need to set task content (we support only text) for example:
What it will prints?
print([[], [], []] * 3)
Then you can write comment, or solution of your task. That`s all finish!)
Yor task will needs approving by our admins, to check status or to see your tasks use /tasks.

Don`t use /timer, please.
